# Tic Tac Doll Project

This is a 3D tic-tac-toe game built with React and Redux, where players can battle against AI（The feature is currently being improved.） or freely control pieces of both sides.
[![Intro of Tic Tac Doll](http://img.youtube.com/vi/ZCWxmSYVd-Q?si=6vqRBIyPIrMIbpU1/0.jpg)](http://www.youtube.com/watch?v=ZCWxmSYVd-Q?si=6vqRBIyPIrMIbpU1 "Tic-Tac-Doll")




## Tech Stack

- React
- Redux
- React Router
- Three.js
- @react-three/fiber
- @react-three/drei
- Express
- Mongoose
- MongoDB

## Installation

Clone the project locally:

```bash
git clone https://github.com/YourGitHubUsername/tic-tac-doll.git
```

## Frontend dependencies
```bash
cd client
npm install
```
## Backend dependencies
```bash
cd server
npm install
```
## Frontend
```bash
cd client
npm start
```
## Backend
```bash
cd server
npm start
```
